# 🎬 Movie Booking App

A modern and user-friendly **Movie Ticket Booking Application** built using Node.js.
This app allows users to browse movies, select show timings, and book tickets easily.

---

## 🚀 Features

* 🎥 Browse available movies
* 🕒 Select show timings
* 💺 Choose seats
* 🎟️ Book tickets instantly
* 📱 Responsive UI
* ⚡ Fast and lightweight

## 📦 Installation

### 🔹 Prerequisites

Make sure you have installed:

* Node.js (v14 or later recommended)
* npm (comes with Node.js)

Check versions:

```bash
node -v
npm -v
```

---

## ▶️ Run Locally

### 1️⃣ Clone the repository

```bash
git clone https://github.com/rgolu1461-bot/movie-ticket-booking.git
```

### 2️⃣ Navigate into the project folder

```bash
cd movie-ticket-booking
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Start the development server

```bash
npm run dev
```

The app will run on:

```
http://localhost:3000
```

(or the port shown in your terminal)
